# 📋 API Lista de Tarefas (Flask + PostgreSQL)

Uma API REST simples para gerenciamento de tarefas, construída com **Flask** e **PostgreSQL**, organizada em arquitetura modular (routes, models e config).

---

# 🚀 Tecnologias

- Python 3
- Flask
- PostgreSQL
- psycopg2
- python-dotenv

---

# Funcionalidades
- Criar tarefas
- Listar tarefas
- Buscar por ID
- Atualizar tarefas
- Deletar tarefas
- Controle de status (feito / pendente)
- Integração com PostgreSQL

# Como Rodar

Instalar dependências
- pip install -r requirements.txt

Criar banco de dados
- Passo a passo no arquivo banco_dados/banco.sql

Configurar .env
- de acordo com suas informaçoes

Rodar projeto
- python app.py
